RDK X5 嵌入式多传感融合竞速小车竞赛技术方案
一、硬件配套需求
主控单元采用地平线 RDK X5 异构开发板，搭载专用 BPU 硬件 AI 加速器；车体采用 OriginCar 阿克曼转向智能底盘，直流驱动电机支持前进、倒车、侧向避让。
传感器硬件：Aurora930 深度 RGB 相机（640×480@30fps）、BMI088 六轴 IMU、500 线增量式轮式编码器；底层运动控制下位机为 STM32，串口通信波特率 921600，I2C 总线传输 IMU 原始数据。
二、环境部署
项目配套自动化部署脚本 install_race_sensor_deps.sh，一键批量部署全栈依赖：ROS2 Humble 环境、地平线 BPU 推理 SDK、OpenCV 图像处理库、robot_localization 多传感器滤波包、二维码识别模型、本地 VLM 视觉大模型。脚本自动授予串口、I2C、深度相机硬件读写权限，无需手动配置底层驱动。
三、项目工程结构
基于标准 ROS2 工作空间 ros2_ws 开发，2026 年 6 月发布，开源协议 Apache 2.0；src 源码目录拆分 5 个低耦合独立功能包，配套参数配置文件、RViz 可视化脚本、节点启动文件：
qr_code_detection：二维码任务识别感知包
imu_odom_fusion：IMU + 轮式编码器定位融合包
yolo_obstacle_detect：YOLOv5s 锥桶障碍物检测包
depth_semantic_perception：深度相机人形立牌语义提取包
track_control_master：赛道分阶段导航总控调度包
四、启动方式
一键完整运行全赛道自动驾驶系统：ros2 launch track_control_master full_race.launch.py
分模块独立调试模式：每个功能包内置专属 launch 启动文件，可单独启动定位、视觉感知、导航控制节点，搭配 rqt_graph、ros2 topic echo 实时观测数据，数据来源统一为深度相机图像流。
五、关键可配置参数
感知检测参数：YOLO 检测置信度 0.3，避障有效像素面积 15000，人形立牌有效测距区间 300~2000mm，防抖滑动窗口 3 帧；
定位融合参数：Mahony 姿态解算比例增益 Kp=0.5，轮速校准系数 X=1.03、Y=1.125；
运动导航参数：最大行驶速度 0.8m/s，绕行最低车速 0.19m/s，预瞄距离 0.15~1.0m，航点定位容差 0.25m，绕行冷却 3s；
安全围栏参数：场地 X 边界 0.1~4.3m，Y 边界 -0.15~4.3m，越界自动低速回退。
六、标准化竞赛四阶段流程
出发巡航阶段：车辆从起点自动出发，沿预设航点行驶，全程开启 YOLO 锥桶实时避障，感知输入统一为深度相机 RGB 图像；
二维码指令读取阶段：抵达二维码任务点位，基于深度相机图像流，双模型解码获取赛道绕行方向，识别超时默认启用顺时针路线；
通道绕行 + 深度语义识别阶段：按扫码指令顺 / 逆时针通行赛道，直道中段启用深度相机分割地面与人形立牌，采集最优帧调用本地 VLM 输出文字描述；
视觉精准停车阶段：返回起点区域，基于深度相机 P 标识单应矩阵测距、峰值面积追踪完成 0.2m 精度定点停车，竞赛流程结束。
七、五大核心功能模块
二维码识别模块
采用微信 CNN 超分增强模型为主、OpenCV 原生检测器降级备用的双识别架构，仅采用深度相机单路图像输入；解码 P1/P2/P3 编号后发布 /qr_code_result 话题，导航节点接收指令切换赛道绕行逻辑，依托深度相机 RGB 画面完成识别，弱光、远距离场景识别容错性强。
IMU 与底盘里程计多传感器融合模块
由 IMU 采集、轮式里程积分、EKF 滤波三节点协同工作；BMI088 陀螺仪 1000Hz 原始采样，Mahony 互补滤波解算稳定航向，扩展卡尔曼滤波融合轮速与惯性数据，输出低漂移融合里程 /odom_combined，作为导航唯一定位数据源，解决无 GPS 室内长距离打滑漂移问题。
YOLOv5s 避障感知模块
依托 RDK X5 BPU 硬件加速推理，基于深度相机 RGB 图像同步识别赛道锥桶、P 型停车标识；四层图像过滤 + 3 帧滑动窗口消除瞬时误检，结合深度信息做距离校验，基于航向误差动态调整通行分割线，搭配电子围栏约束自适应生成平滑绕行轨迹，绕行过程禁止倒车。
深度相机截取人物立牌模块
利用深度相机深度值区分地面与 30mm 以上凸起立牌，剔除过近、过远干扰区域，图像降噪后筛选尺寸合规目标，选取画面面积最大人形立牌；间隔 3 秒采集 3 帧深度相机图像，冷却 30 秒后调用本地 VLM 生成语义描述，通过 ROS 话题下发主控系统。
多阶段惯性导航主控模块
50Hz 高速控制循环，搭载改进 Pure Pursuit 纯跟踪算法，动态滤波预瞄距离、弯道曲率自适应限速、远近端 PID 姿态平滑过渡；有限状态机管理四阶段赛事，控制优先级：电子围栏＞扫码姿态校正＞YOLO 避障＞视觉停车＞基础航点导航。
